# PokemonApp

## General info
This project is simple React based app for searching and browsing pokemons from PokeAPi.

## Technologies
Project is created with:
* React
* Axios
* PokeAPI from https://pokeapi.co

## Setup
To run this project, install it locally using npm:

```
$ cd ../PokemonApp-main
$ npm install
$ npm run dev
```
